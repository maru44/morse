# morse
Type Morse code!

## demo

![demo](https://raw.github.com/wiki/maru44/morse/images/morse.gif)

## how to start

get package needed

`go mod tidy`

Ignition!

`go run morse.go`

## Settings

You can edit settings by editing config/settings.go.<br/>
I will introduce the default settings.

```
"j" > . (dot)
"k" > - (dash)   (You don't have to press it long.)
"l" > to quit it

0.4sec > space
```
